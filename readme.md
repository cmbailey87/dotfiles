## Usage

1. Prompt, bash completion, bash aliases, etc. (Each time you make changes, just run this command again.)

    ```bash
    ./bootstrap.sh
    ```
2. Homebrew packages.

    ```bash
    ./brew.sh
    ```
3. OSX defaults.

    ```bash
    ./.osx
    ```
